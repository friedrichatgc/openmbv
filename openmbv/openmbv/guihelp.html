GUI Help

Keyboard Interaction

Global Shortcuts

Many actions in the menus have assigned a keyboard shortcut. The shortcut key is shown on the right side of each menu entry.

Beside these also the following keyboard shortcuts exist:

Up or K
Go one frame up
Down or J
Go one frame down
Page-Up
Increase animation speed
Page-Down
Decrease animation speed
1, 2, 3, 4, 5, 6, 7, 8 or 9
Expand object list to depth

Object List Shortcuts

F2
Edit the name of the current object list entry

Mouse Interaction

All GUI elements behave like common GUI elements: try mouse and touch-screen events.

3D-View

In the 3D view the following mouse and touch screen events exist:

Mouse Click Events

Left-Click
Select the object under the cursor
Right-Click
Select the object under the cursor and show its the context menu
Ctrl-Down + Left-Click
Toggle the selection of the object under the cursor
Ctrl-Down + Right-Click
Show context menu for selected objects
Alt-Down + Left-Click
If more then one object is under the cursor show a list of these objects before starting the Left-Click action
Alt-Down + Right-Click
if more then one object is under the cursor show a list of these objects before starting the Right-Click action
Middle-Click
Seeks the focal point of the camera to the point on the shape under the cursor
Left-Double-Click
Open the property dialog of the object under the cursor

Mouse Move Events

Left-Down + Move
Rotate the scene around the axis in the screen plane
Ctrl-Down + Left-Down + Vertical-Move
Rotate the scene in the screen plane
Right-Down + Move
Translate the scene in the screen plane
Middle-Down + Vertical-Move
Zoom the scene
Ctrl-Down + Middle-Down + Vertical-Move
Change the focal distance of the camera while preserving the zoom (only relevant for perspective cameras)
Wheel-Up/Down
Increase/decrease the current frame number
Ctrl-Down + Wheel-Up/Down
Move cursor position in screen z-direction (mainly only useful in stereo view)

Touch-Screen Tap Events

Tap
Select the object under the cursor
Tap-and-Hold
Select the object under the cursor and show its the context menu
Ctrl-Down + Tap
Toggle the selection of the object under the cursor
Ctrl-Down + Tap-and-Hold
Show context menu for selected objects
Alt-Down + Tap
If more then one object is under the cursor show a list of these objects before starting the Left-Click action
Alt-Down + Tap-and-Hold
if more then one object is under the cursor show a list of these objects before starting the Right-Click action
Double-Tap
Open the property dialog of the object under the cursor

Touch-Screen Move Events

1-Tap-Pan
Rotate the scene around the axis in the screen plane
2-Tap-Pan
Translate the scene in the screen plane
2-Tap-Pitch
Zoom the scene
2-Tap-Rotate
Rotate the scene in the screen plane (the two taps must rotate a large angle to start this event)
Ctrl-Down + 2-Tap-Pitch
Change the focal distance of the camera while preserving the zoom (only relevant for perspective cameras)
