Mouse Interaction

All GUI elements behave like common GUI elements: try left, right and double-clicks.

3D-View

Mouse Move Events

Left-Button-Down + Move
Rotate the entiry scene
Right-Button-Down + Move
Translate the entiry scene
Middle-Button-Down + Move
Zoom the entiry scene
D-Down + Left-Button-Down + Move
Manipulate a dragger. The dragger must be enabled using the property menu of the body before
D+Shift-Down + Left-Button-Down + Move
Manipulate a dragger in constraint motion. The dragger must be enabled using the property menu of the body before

Mouse Click Events

Left-Button-Click
Select the object under the cursor
Right-Button-Click
Select the object under the curser and show its the property menu
Ctrl-Down + above action
Toggle the selection of the object under the cursor instead of selecting
Alt-Down + above action
Shows a menu of all objects under the cursor instead of using the first one
Middle-Button-Click
Seeks the focal point of the camera to the point on the shape under the cursor
